<task type="auto">
  <name>Task 2: OCRPipeline orchestrator and BoardStatePoller integration</name>
  <files>src/main/ocr/OCRPipeline.ts, src/main/overlay/BoardStatePoller.ts, src/main/overlay/BoardStateParser.ts</files>
  <action>
1. Create `src/main/ocr/OCRPipeline.ts`:
   - Class that owns: ScreenCapturer, ShopOCR, BoardOCR, ChampionMatcher instances
   - `async initialize(champions: Champion[]): Promise<void>` — creates ChampionMatcher from champion list, initializes ShopOCR Tesseract worker, shares worker with BoardOCR
   - `async tick(): Promise<OCRResult>` — single-tick orchestration:
     a. Capture screenshot via ScreenCapturer (window title from constant)
     b. If capture returns null, return offline result
     c. Check shop visibility via isShopVisible
     d. If shop visible: run ShopOCR.readShop
     e. Run BoardOCR.readBench and BoardOCR.readBoard
     f. Compute owned flag on shop slots: for each ShopSlot, set owned = true if apiName matches any champion in board or bench arrays
     g. Return OCRResult with ocrStatus: 'active'
   - `async terminate(): Promise<void>` — cleanup Tesseract worker
   - Stale data tracking: store `lastValidResult` and `lastValidAt` timestamp
     - If tick() capture returns null: check if (Date.now() - lastValidAt) < 10000 → return lastValidResult with ocrStatus: 'stale'
     - If > 10 seconds of null: return empty result with ocrStatus: 'offline'

2. Update `src/main/overlay/BoardStatePoller.ts`:
   - Add OCRPipeline as a dependency (create in constructor or via setter)
   - In `start()`: call `ocrPipeline.initialize(champions)` (champions passed as parameter or loaded via IPC)
   - In `tick()`: call both `fetchGameData()` AND `ocrPipeline.tick()` in parallel
   - Merge results: parseOverlayState for API data + OCR result fields
   - Send merged OverlayState via IPC

3. Update `src/main/overlay/BoardStateParser.ts`:
   - Update `parseOverlayState` to accept optional OCRResult parameter
   - When OCRResult provided, merge its fields into the OverlayState
   - When OCRResult not provided, use defaults: board: [], bench: [], shop: [], shopVisible: false, ocrStatus: 'offline'
  </action>
