  <name>Task 2: ShopOCR and ShopVisibilityDetector</name>
  <files>src/main/ocr/ShopOCR.ts, src/main/ocr/ShopVisibilityDetector.ts, src/main/ocr/ShopOCR.test.ts, src/main/ocr/ShopVisibilityDetector.test.ts</files>
  <behavior>
    - ShopOCR.initialize() creates a persistent Tesseract.js worker with eng language and letter+space whitelist
    - ShopOCR.readShop(pngBuffer, width, height, champions) returns ShopSlot[] with 5 entries
    - Each ShopSlot has apiName (from fuzzy match) or null if unrecognized
    - ShopOCR.terminate() cleans up the Tesseract worker
    - isShopVisible(pngBuffer, width, height) returns true when shop region pixel is dark (R<30, G<30, B<30)
    - isShopVisible returns false when shop region pixel is bright (combat/carousel)
  </behavior>
  <action>
1. Create `src/main/ocr/ShopVisibilityDetector.ts`:
   - Export `async function isShopVisible(pngBuffer: Buffer, width: number, height: number): Promise<boolean>`
   - Uses Jimp to read the pixel at the SHOP_REGION landmark coordinate (scaled by width/1920, height/1080)
   - Returns true if pixel color is dark (R < 30 AND G < 30 AND B < 30)
   - Import SHOP_REGION from OCRCoordinates

2. Create `src/main/ocr/ShopOCR.ts`:
   - Class with `async initialize(): Promise<void>` — creates persistent Tesseract.js worker with eng, PSM.SINGLE_LINE, letter+space whitelist. Runs a warmup recognize call.
   - `async readShop(pngBuffer: Buffer, width: number, height: number, matcher: ChampionMatcher): Promise<ShopSlot[]>`
     - Computes scale factors: scaleX = width / 1920, scaleY = height / 1080
     - For each of 5 SHOP_SLOT_CENTERS: compute scaled name band coordinates (center + offset), crop using cropRegion, run worker.recognize on cropped buffer
     - Fuzzy-match recognized text via matcher.match()
     - If confidence < 60 or matcher returns null, set apiName = null
     - Return 5 ShopSlots (apiName, cost from matcher result, owned: false — owned flag set later during integration)
   - `async terminate(): Promise<void>` — terminates worker

3. Create tests:
   - ShopVisibilityDetector test: create a tiny synthetic dark image (all black pixels) and verify returns true. Create a bright image and verify returns false. Mock Jimp or use actual Jimp with synthetic buffers.
   - ShopOCR test: mock tesseract.js worker (vi.mock('tesseract.js')), verify initialize creates worker with correct params, verify readShop calls recognize for each slot and passes results through ChampionMatcher. Use a mock ChampionMatcher that returns predictable results.
  </action>
