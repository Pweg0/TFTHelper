Create `src/main/startup.ts` — the main startup sequence:

`runStartupSequence(win: BrowserWindow): Promise<void>`:
1. Initialize electron-store with AppConfig schema (patchVersion, userLocale, userRegion, metaScrapedAt, metaScrapedPatch).
2. Send status to renderer: "Checking for updates..."
3. Call `getLatestPatchVersion()`. If fails, log warning and use cached version from store.
4. Call `isCacheStale(store)`. If stale:
   a. Send status: "Downloading game data..."
   b. Determine locale — for now default to "en_us" (locale auto-detection from LCU API is a Phase 1 open question; use default with a TODO comment noting it should detect from Riot client process args `--locale=pt_BR`).
   c. Determine current set — for now hardcode the current set identifier with a TODO (e.g., "set13" or whatever the CommunityDragon data reveals). This will be refined when the actual JSON structure is inspected.
   d. Call `downloadStaticData(locale, set)`.
   e. Update store: patchVersion = liveVersion.
5. Send status: "Loading meta builds..."
6. Call `refreshMetaIfStale(store, currentPatch)`. Wrap in try/catch — if fails, log and continue.
7. Send status: "Ready! Waiting for TFT game..."
8. Create GameWatcher with callbacks:
   - onGameStart: send 'game-started' IPC event to renderer with game data
   - onGameEnd: send 'game-ended' IPC event to renderer
9. Start GameWatcher polling.

The status updates are sent via `win.webContents.send('startup-status', { step, message })`.

Create `src/main/ipc/handlers.ts`:

`registerIpcHandlers(store: Store): void`:
- `ipcMain.handle('get-static-data')` — read and return cached static data from disk
- `ipcMain.handle('get-meta-data')` — read and return cached meta data from disk
- `ipcMain.handle('get-config')` — return { patchVersion, userLocale, userRegion } from store

Update `src/preload/preload.ts`:
```typescript
import { contextBridge, ipcRenderer } from 'electron';

contextBridge.exposeInMainWorld('api', {
  getStaticData: () => ipcRenderer.invoke('get-static-data'),
  getMetaData: () => ipcRenderer.invoke('get-meta-data'),
  getConfig: () => ipcRenderer.invoke('get-config'),
  onStartupStatus: (callback: (status: { step: string; message: string }) => void) =>
    ipcRenderer.on('startup-status', (_event, status) => callback(status)),
  onGameStarted: (callback: (data: unknown) => void) =>
    ipcRenderer.on('game-started', (_event, data) => callback(data)),
  onGameEnded: (callback: () => void) =>
    ipcRenderer.on('game-ended', () => callback()),
});
```

Update `src/main/index.ts`:
- On app.whenReady(): create BrowserWindow, load renderer.
- Call `registerIpcHandlers(store)`.
- Call `runStartupSequence(win)` after window is ready.
- Handle app.on('window-all-closed') for quit.
