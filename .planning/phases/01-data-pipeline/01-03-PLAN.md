Create `src/main/game/GameWatcher.ts`:
- Class `GameWatcher` with constructor taking `{ onGameStart: (data: LiveClientResponse) => void, onGameEnd: () => void, pollIntervalMs?: number }`.
- Default poll interval: 3000ms (3 seconds per research recommendation).
- Internal state: `isGameActive: boolean` (starts false).
- `start(): void` — starts `setInterval` calling `fetchGameData()`. On each tick:
  - If data is not null AND `data.gameData.gameMode === 'TFT'` AND `!isGameActive`: set isGameActive=true, call onGameStart(data).
  - If (data is null OR gameMode !== 'TFT') AND isGameActive: set isGameActive=false, call onGameEnd().
  - Otherwise: no action (stable state).
- `stop(): void` — clears the interval.
- `getState(): GameState` — returns current state snapshot.
