  <name>Checkpoint: Live TFT game overlay verification</name>
  <files></files>
  <action>
    Human verification of complete overlay system with a live TFT game.

    Before testing, ensure:
    1. TFT is set to Windowed Fullscreen mode (Settings > Video > Window Mode). Exclusive fullscreen will NOT work with electron-overlay-window.
    2. Run the app: npm run dev (or npx electron-vite dev)
    3. Start a TFT match (normal, ranked, or Hyper Roll all work)

    Step 1 — Overlay visibility:
    - Start a TFT game while the app is running
    - Verify the overlay panel appears on the right side of the screen over the TFT window
    - If overlay does NOT appear: check the console/terminal for "attach" event logs. The TFT window title may need to be corrected.

    Step 2 — Click-through:
    - Click on the game area (not the overlay panel) — verify clicks pass through to TFT
    - Move your mouse over the overlay panel — verify the panel becomes interactive (no click-through on panel)
    - Move mouse away from panel — verify click-through resumes

    Step 3 — Board state data:
    - During a match, verify player names appear in the overlay
    - Check if HP, level, and champion info display correctly or show "?" fallbacks
    - Verify eliminated players disappear from the list
    - Verify players are sorted by HP (highest at top)
    - Check the terminal/console for raw API response logs — this reveals the actual TFT field names

    Step 4 — Game end behavior:
    - When a game ends (you are eliminated or win), verify the last board state remains visible
    - Verify it stays visible until you start a new game

    Step 5 — Capture API response:
    - Copy the raw Live Client API response logged to the terminal during the first few polling ticks
    - This is needed to correct any provisional type mappings

    Report: What worked, what failed, and paste the raw API response if field mappings were incorrect.
  </action>
  <verify>Human confirms overlay behavior during live TFT game</verify>
  <done>Overlay verified working with live TFT game, or issues documented with raw API response for correction</done>
</task>
