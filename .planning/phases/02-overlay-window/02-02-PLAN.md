  <name>Task 1: Install electron-overlay-window and create OverlayWindow module</name>
  <files>package.json, src/main/overlay/OverlayWindow.ts, src/main/overlay/OverlayWindow.test.ts</files>
  <action>
    1. Install electron-overlay-window:
       ```
       npm install electron-overlay-window
       ```
       If native addon fails, also run: `./node_modules/.bin/electron-rebuild -f -w electron-overlay-window`
       Add `electron-rebuild` as a devDependency if not present.

    2. Create `src/main/overlay/OverlayWindow.ts`:
       - Import `OverlayController`, `OVERLAY_WINDOW_OPTS` from `electron-overlay-window`
       - Import `BrowserWindow`, `ipcMain` from `electron`
       - Export `createOverlayWindow()` function that:
         a. Creates a BrowserWindow with `...OVERLAY_WINDOW_OPTS` spread, plus webPreferences with preload pointing to `../preload/overlayPreload.js` and sandbox: false
         b. Loads the overlay HTML: in dev mode use `process.env['ELECTRON_RENDERER_URL'] + '/overlay.html'`, in prod use `join(__dirname, '../renderer/overlay.html')`
         c. Calls `OverlayController.attachByTitle(overlayWin, 'League of Legends (TM) Client')` â€” IMPORTANT: attachByTitle can only be called ONCE per process. The overlay window must NEVER be destroyed.
         d. Listens for OverlayController events:
            - `attach`: call `overlayWin.show()`
            - `detach`: call `overlayWin.hide()`
            - `moveresize`: call `overlayWin.setBounds({ x, y, width, height })`
         e. Sets initial click-through: `overlayWin.setIgnoreMouseEvents(true, { forward: true })`
         f. Returns the BrowserWindow instance
       - Register IPC handler `set-ignore-mouse-events` on ipcMain.on (not handle) that calls `BrowserWindow.fromWebContents(event.sender)?.setIgnoreMouseEvents(ignore, { forward: true })` â€” this enables toggling click-through from the overlay renderer on panel hover
       - IMPORTANT: Add `electron-overlay-window` to the `externalizeDepsPlugin` exclude list in electron.vite.config.ts main plugins, since it's a native addon that must not be bundled

    3. Create `src/main/overlay/OverlayWindow.test.ts`:
       - Mock `electron` (BrowserWindow, ipcMain) and `electron-overlay-window` (OverlayController, OVERLAY_WINDOW_OPTS)
       - Test that createOverlayWindow creates a BrowserWindow with OVERLAY_WINDOW_OPTS spread
       - Test that attachByTitle is called with correct window title
       - Test that setIgnoreMouseEvents(true, { forward: true }) is called on creation
       - Test that the set-ignore-mouse-events IPC handler registers and calls setIgnoreMouseEvents on the correct window
  </action>
